# MemTypeTool
MemType Tool - GUI For managing the [MemType Offline password keeper](https://github.com/jim17/memtype)

##Dependencies
Make sure you have `python 3` and `PyQt4` installed.

##Running
To run the **MemTypeTool**:
`python main.py`
